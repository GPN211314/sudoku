# sudoku.exe [选项] 参数

## 选项：

####     -c <数字>	生成<数字>个数独终局至文件sudoku.txt

####     -s <绝对路径>	从<绝对路径>中读取数独题目并生成一个可行解至sudoku.txt

####     -h 显示当前帮助信息
